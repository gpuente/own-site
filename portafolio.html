ASCOMED

Sistema web para la administración y gestion de presupuestos médicos, desarrollado para la empresa ASCOMED.

El sistema permite el mantenimiento de datos maestros correspondientes a Pacientes, Médicos, Códigos de Cirugías, Isapres, Especialidades y Bancos. Adicionalmente el sistema permite la gestión de presupuestos médicos, cirugías y pagos.

EL flujo de trabajo se inica creando un presupuesto de cirugía en el módulo "Presupuestos" para un paciente específico, al cual se le asigna un equipo médico a cargo de la cirugía. Posteriormente, en caso de que el paciente apruebe el presupuesto, se procede a cerrar el mismo por medio del módulo "Cirugías", en dónde se asigna el equipo médico definitivo. Finalmente, Se registran todos los pagos y anotaciones de cada cirugía en el módulo de "Pagos"

Presupuestos, Cirugías y Pagos poseen una versión imprimible, la cual es entregada al paciente. Adicionalmente, cada uno de estos módulos posee una versión descargable en ".xls".

El sistema permite la emisión de reportes para deudas asignadas a un paciente específico y para las cirugías asignadas a un médico en particular. Ambos reportes son exportados a ".xls"

La aplicación web fue construida utilizando tecnologías PHP, CSS, Bootstrap, JavaScript, JQuery, MySQL, AdminLTE y AJAX, utilizando el patrón de diseño MVC por medio de un framework desarrollado específicamente para la aplicación. Para brindar una mejor experiencia de usuario, se implementaron tecnologías de Responsive Design, con lo cual el contenido del sitio se adapta al tamaño de la pantalla en dónde es visualizado, permitiendo trabajar así desde Tablets, SmartPhones o PC's.
